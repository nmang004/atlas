# Glossary of Terms

Quick definitions of common SEO and content terms used throughout this handbook.

## A-D

| Term | Definition |
|------|------------|
| **Above the Fold** | Content visible without scrolling when a page first loads |
| **AI Overview** | Google's AI-generated summary that appears at the top of some search results |
| **Anchor Text** | The clickable text of a hyperlink |
| **Canonical Tag** | HTML element that tells search engines which version of a page is the primary version |
| **Click-Through Rate (CTR)** | Percentage of people who click a link after seeing it in search results |
| **Conversion Velocity** | How quickly a user can move from interest to taking action (calling, submitting a form) |
| **CTA** | Call to Action—the element prompting users to take a specific action |
| **Dwell Time** | How long a user stays on a page before returning to search results |

## E-H

| Term | Definition |
|------|------------|
| **E-E-A-T** | Experience, Expertise, Authoritativeness, Trustworthiness—Google's quality framework |
| **Featured Snippet** | A highlighted answer box that appears at the top of some search results |
| **H1, H2, H3** | HTML heading tags indicating hierarchy (H1 is most important) |
| **Hub and Spoke** | Content structure where a main topic page (hub) links to related subtopic pages (spokes) |

## I-N

| Term | Definition |
|------|------------|
| **Internal Link** | A link from one page on a domain to another page on the same domain |
| **Local Pack** | The map and business listings that appear in local search results |
| **Meta Description** | HTML tag that provides a brief summary of a page, often displayed in search results |
| **Money Page** | A high-value page designed to convert visitors (typically a Service Page) |
| **NAP** | Name, Address, Phone number—the core local business information |

## O-S

| Term | Definition |
|------|------------|
| **Orphan Page** | A page with no internal links pointing to it |
| **People Also Ask (PAA)** | Related questions Google displays in search results |
| **Schema Markup** | Structured data code that helps search engines understand page content |
| **SERP** | Search Engine Results Page—what appears when you search on Google |

## T-Z

| Term | Definition |
|------|------------|
| **Time to Value (TTV)** | How quickly a visitor can find what they need after landing on a page |
| **Title Tag** | HTML element that defines the title of a page, displayed in search results and browser tabs |
| **YMYL** | Your Money or Your Life—topics with potential impact on health, finances, or safety |
| **Zero-Click Search** | A search where the user gets their answer directly in search results without clicking |
