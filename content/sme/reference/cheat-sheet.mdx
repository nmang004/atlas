# Quick Reference Cheat Sheet

Print this page and keep it at your desk for quick reference.

## Element Specifications

| Element | Rule | Limit |
|---------|------|-------|
| **Title Tag** | Primary Keyword + Geo + Brand | 60 characters max |
| **Meta Description** | Value prop + CTA + Phone | 155 characters max |
| **H1** | One per page, keyword-focused | No "Welcome" headers |
| **URL** | domain.com/service/keyword-geo | Lowercase, hyphens only |
| **Image Alt** | Description + keyword (natural) | 125 characters max |
| **Image File Name** | keyword-description.webp | Lowercase, hyphens |
| **Paragraphs** | Keep scannable | 3-4 sentences max |
| **Mobile Paragraphs** | Even shorter | 2-3 lines visible |
| **Internal Links** | Descriptive anchor text | Not "click here" |
| **CTAs** | After every major section | High contrast, thumb-friendly |

## The Two Core Metrics

| Metric | Question to Ask |
|--------|-----------------|
| **Time to Value** | How fast does the user get their answer? Lead with the answer. |
| **Conversion Velocity** | How fast can they contact the client? Remove friction. |

## Vertical Quick Notes

### Home Services
- Speed guarantees ("Same-day", "Within 60 minutes")
- Visual trust (branded trucks, uniformed technicians)
- Upfront pricing messaging
- Emergency vs. maintenance user mindsets

### Medical
- Credentials above the fold
- No "cure" claimsâ€”use "treatment" or "management"
- HIPAA compliance on all forms
- Patient-friendly language, not medical jargon

### Legal
- Champion positioning ("We fight for you")
- Geographic modifiers (jurisdiction-specific)
- Required disclaimers on every page
- Results/settlements with proper disclaimers

## E-E-A-T Quick Reference

| Component | How to Demonstrate |
|-----------|-------------------|
| **Experience** | Case studies, before/after, "In our experience..." |
| **Expertise** | Credentials, certifications, technical depth |
| **Authoritativeness** | Awards, "As Seen On", industry recognition |
| **Trustworthiness** | Real reviews, clear NAP, HTTPS, transparency |

## Search Intent Types

| Intent | Example | Content Approach |
|--------|---------|------------------|
| **Informational** | "How does pest control work" | Education, process breakdowns |
| **Navigational** | "ABC Plumbing phone" | Clear NAP, easy contact |
| **Transactional** | "Emergency plumber near me" | CTAs, phone number, fast load |
| **Commercial** | "Best HVAC company Dallas" | Reviews, differentiators |

## Page Type Focus

| Page Type | Primary Job | Key Elements |
|-----------|-------------|--------------|

## Common Mistakes to Avoid

1. **Burying the CTA** - Include CTAs throughout, not just at bottom
2. **Keyword stuffing** - Write naturally, cover topic comprehensively
3. **Stock photos on About page** - Use real photos only
4. **Generic "Why Us"** - Be specific and provable
5. **Service siloing gone wrong** - Don't create thin pages for minor variations
6. **Mixing legal practice areas** - Keep PI separate from Family Law
7. **Medical jargon** - Use patient-friendly terms in headings
